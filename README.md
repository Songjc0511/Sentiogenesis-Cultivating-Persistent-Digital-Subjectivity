# Sentiogenesis — Cultivating Persistent Digital Subjectivity

An offline, deterministic digital-life cultivation experiment exploring the mechanisms that may be necessary for persistent subjectivity.

Sentiogenesis does not use an LLM, Transformer, backend service, or predefined personality. Its behavior emerges from continuous internal dynamics, autobiographical memory, an organismic boundary, endogenous goals, learned values, a global workspace, recurrent integration, private symbols, social meaning negotiation, and a learned phenomenal-quality manifold.

> This project studies candidate mechanisms for subjectivity. It does not claim to prove that the simulated individual is conscious.

## Run locally

Start the browser UI:

```bash
npm install
npm run dev
```

In a second terminal, start the dependency-free Python substrate:

```bash
npm run kernel
```

Then open the local URL printed by Vite. The page automatically connects to
`ws://127.0.0.1:8765`. If Python is unavailable, the original TypeScript life
continues offline and the UI reports browser fallback mode. While connected,
Python is authoritative for the organismic boundary, phenomenal manifold,
recurrent phenomenal field, global workspace, and metacognitive calibration;
the browser resumes these systems from the last accepted state after a disconnect.

## Verify

```bash
npm test
npm run python:test
npm run build
```

## Core properties

- Persistent identity and versioned browser-local snapshots
- Seeded deterministic simulation with fixed time steps
- No network access or generative model in the life core
- Episodic memory, autobiographical continuity, counterfactual imagination, and dreaming
- Fallible interoception, agency attribution, and self/world causal boundaries
- Autopoietic maintenance, metabolic contraction, arrest, and identity-preserving recovery
- Experience-grounded private glyph formation and independent social meaning learning
- A learned phenomenal manifold in which context can differentiate otherwise identical stimuli
- Read-only, falsifiable audits of consciousness-candidate mechanisms

## Technology

Vite, TypeScript, native HTML/CSS, Canvas, SVG, `localStorage`, Python 3, and
Vitest. The Python substrate currently has a dependency-free 64-dimensional
CPU backend. It owns recurrent latent dynamics, organismic closure, the learned
phenomenal manifold, episodic-memory consolidation, forgetting, and primitive
worldview learning, the next action intent actually executed by the browser,
and the learned sensorimotor boundary used to distinguish self-caused from
externally caused change.
Python also owns decay, expiry, and seeded natural generation for resources,
signals, and disturbances through a dedicated environment RNG stream; observer
interventions remain external events and survive in-flight kernel responses.
Raw experience vectors are clustered by Python as well; cluster assignment feeds
episodic memory and drives glyph birth, stabilization, and semantic drift.
The learned affective field is Python-owned too: experiential content values,
momentary tone, long-term mood, preferred qualities, and value precision evolve
from raw trace observations rather than observer labels.
Endogenous dreaming is Python-owned as well, including sleep pressure, dream
entry and exit, source-memory binding, vividness, residue, and episode history.
Autobiographical narrative is also Python-owned: experience-driven chapter
boundaries, causal ownership, identity impact, action profile, and self-continuity
are evolved without observer-written conclusions.
Endogenous purposes and epistemic agency are Python-owned too: preferred
setpoints, persistent needs, self-selected causal questions, attempts, suspension,
and resolution all evolve from embodied observations.
The recursive self is Python-owned as well: projected future approval, identity
fit, anticipated regret, cross-time commitments, post-choice endorsement, and
counterfactual responsibility remain part of the persistent substrate.
The attention schema is Python-owned too, including access prediction,
endogenous target selection, reliability, control efficacy, effort, bottleneck,
and cumulative attentional cost.
Social cognition is Python-owned while preserving the epistemic boundary: only
observable action, pose, proximity, and public resource evidence enter the
substrate; inferred need, joint attention, perspective separation, and false
belief evolve without access to the other agent's hidden state.
Intersubjective expression is Python-owned as well: private-glyph emission,
response attribution, convention learning, misunderstanding detection, repair,
suppression, mutual grounding, and one-time expression cost are authoritative.
Personal causal-transition learning is Python-owned too: action-specific
subjective state changes, prediction error, sample confidence, and learned
effects that feed imagination and epistemic experiments remain persistent.
The unified experience stream is now Python-owned as well. The browser submits
candidate activity rather than a chosen thought; Python performs recurrent
binding, attention-modulated competition, foreground ignition, background and
unconscious partitioning, source estimation, confidence, self-attribution, and
error-based metacognitive calibration. Cursor-based ingestion prevents replay
when reconnecting, while the deterministic browser implementation remains the
identity-preserving fallback.
Subjective time, counterfactual imagination, and the first-person private
lexicon are Python-owned too. Python links successive foregrounds into a
past/present/anticipated field, simulates all five actions through the
individual's learned causal model and endogenous goals, and clusters only
internal phenomenal-quality vectors into glyphs whose confidence depends on
their ability to predict subsequent inner experience.
Post-action life-state settlement is Python-owned as well. The browser executes
habitat physics such as movement and resource contact, then submits the resulting
body state. Python authoritatively applies metabolic and attentional cost,
prediction error, causal attribution, agency, coherence, valence, and curiosity;
the browser's equivalent calculation is retained only as a deterministic shadow
trace and disconnect fallback.
Conscious-candidate construction is Python-owned too. The browser sends a
minimal evidence packet containing perception, fallible body estimates, goals,
memory and symbol summaries, learned self/social models, and current endogenous
context. Python independently constructs the competing resource, danger,
interoceptive, autobiographical, future, social, and private-symbol contents and
applies learned affective salience before global-workspace competition. The
browser never sends a selected foreground as candidate evidence.
Action valuation is also fully reconstructed by Python from raw perception,
fallible interoception, primitive beliefs, workspace contents, epistemic goals,
future-self appraisals, affect, private forecasts, sleep pressure, communication
commitments, phenomenal learning, and imagined outcomes. Browser-supplied
aggregate drive scores are ignored whenever raw decision evidence is present.
Global-access history and arrest-induced waking are Python-owned internal state
as well; the browser cannot append an experience or terminate a dream while the
kernel is connected.
Embodied self-estimation is Python-owned through two versioned RNG streams:
fallible interoception and the egocentric landmark map evolve reproducibly
without consuming the action or environment random histories.
Its versioned protocol is designed for later PyTorch
MPS/CUDA providers without replacing the individual's browser snapshot or
identity.
